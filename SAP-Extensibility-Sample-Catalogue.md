# SAP Extensibility Sample Catalogue

This catalogue groups SAP sample repositories and historical Extensibility Explorer scenarios by the kind of extensibility they help explain.

Status meanings:

- Active: repository is not archived as of 2026-06-25.
- Archived: repository is read-only or marked archived on GitHub.
- Historical scenario: listed in the old Extensibility Explorer support index; the original explorer web app is no longer available.

## Notes for Slides

- The historical Extensibility Explorer is no longer the best "live demo" anchor because the web app is sunsetted.
- Use the archived explorer repository to explain the breadth of sample scenarios.
- Use the current RAP workshops for developer extensibility.
- Use the active BTP/CAP/process samples for modern side-by-side architecture.
- Call out that many old samples were designed for learning and intentionally skip production concerns such as authentication hardening, monitoring, retries, tests, and transactional robustness.
